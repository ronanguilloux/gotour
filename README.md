GOTOUR
======


Documentation
-------------

The main tutorial is here : [tour.golang.org](http://tour.golang.org)


Build & run
-----------

Install golang, then use the go CLI:

    go install gotour/exercices
    go install gotour/main
    bin/main

Or just run this: (it's the same)

    bin/build
